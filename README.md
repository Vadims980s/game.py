# game.py
A game I've made using **pygame** (Python).
In this game, you can move, jump (through lava), collect coins and pass through a few levels.
Right now, there are only five levels.

## How to install
1. Clone the respitory
2. Open up Command Prompt and open up Python
3. Install the following libraries (using <code>pip install</code>):

**Pygame**

4. Launch the Python file
5. Now the game should work!
